# 镜头设计（shot-design）— 山音融合

> 摘自山音超级导演大师 `shot-design.md` 和 `core-methodology.md` 第一节，
> 适配本仓库的 Wan2.7 r2v / i2v / t2v 实战。

主 SKILL.md 已经讲过续接黄金五条 + 模型选型 + 续帧决策树（**工程层**，
不要违反）。本文件讲**艺术层**：每个 shot 的叙事目的、镜头组组合、
对白镜头的非套路化设计。

---

## 一、叙事目的分析法（narrative_purpose）

每个 shot 的存在价值用一句话回答：「如果这一镜不存在，故事会丢什么？」

### 合格的叙事目的（具体到视听手段）

- 「用低角度仰拍 + 缓慢推近，放大钱夫人挑衅时的优越感」
- 「通过她偷瞄郭芙蓉的眼神，暗示她已经心虚」
- 「从笑意到僵硬，完成面子→错愕的内部节奏切换」
- 「用大特写指尖颤抖，把人物的虚张声势暴露给观众」

### 黑名单（VFX Reviewer 会拦下）

| 空话 | 为什么不行 |
|------|-----------|
| 展现冲突 / 推进剧情 | 没有视听落点 |
| 建立场景 / 渲染气氛 | 不说用什么手段建立、渲染什么 |
| 表现情绪 | 不说哪种情绪，怎么表现 |
| TBD / TODO | 占位符 |

### 写法模板

「用 **<具体视听手段>** + **<具体动作 / 表情 / 物体>**, 达成
**<故事 / 节奏 / 信息 / 情绪>** 上的 **<具体效果>**」

例：「用 *低角度仰拍 + 缓推* 配合 *钱夫人抬下巴轻笑*, 在 *外部冲突
推进* 的同时让观众感到 *她虚张声势的优越感*」

## 二、镜头组（shot group）

把 3-5 个紧密相接、共同完成一个叙事单元的 shot 编为一组，组内分工：

| 组型 | 何时用 | 组员构成 |
|------|--------|----------|
| **蒙太奇组** | 用一组并列镜头建立氛围 / 主题 | 全是建立 / 对比 |
| **递进组** | 强度逐步上升直到爆发 | 建立 → 递进 → 递进 → 收尾 |
| **因果组** | 一个动作引发另一个反应 | 建立(因) → 反应(果) |
| **对比组** | 把矛盾两面切来切去 | 建立 → 对比 → 建立 → 对比 |

### shot_group_role 含义

- `建立` —— 引入这组的核心元素。通常是组内第一镜或最重要的全景。
- `递进` —— 推升强度。通常是中景或近景，加重表情或动作幅度。
- `反应` —— 切到承受方 / 对照方，展示影响。
- `对比` —— 蒙太奇式对照，提供另一个视角。
- `收尾` —— 这一组的高点或结点，常常是特写或大全景。

### 同组镜头的工程要求

1. 共享 `seed`（同 scene 已经共享一个 seed，组内不要再换）。
2. mood_anchor 必须每个 shot 都拼到末尾（铁律不变）。
3. **续帧决策**：组内连续 shot 间 `use_prev_last_frame_as_first=true`，
   组与组之间也可以 true（除非跨 scene）。

## 三、动作-反应结构

电影叙事的最小单元是「**动作 → 反应**」，而不是单个动作。

每当你写一个动作 shot，问自己：「下一镜是不是反应？」如果不是，
观众会觉得动作"没着落"。

例：
- 动作镜：钱夫人抬手指着郭芙蓉
- 反应镜（必须有）：郭芙蓉冷眼回视 / 旁观者倒吸一口气 / 钱夫人自己
  突然手抖

**铁律**：高潮场景里，每个关键动作 shot 之后必须有一个反应 shot。
没有反应的动作 = 没有意义的动作。

## 四、切镜连接逻辑

相邻两镜之间的视觉跳跃必须有逻辑。三类合法的连接方式：

### 1. 景别阶梯连接（Match cut by scale）

- 全 → 中 → 近 → 特：从外到内，节奏渐紧
- 特 → 近 → 中 → 全：从内到外，节奏松开
- **跳级连接**（全 → 特、特 → 全）只在强烈情感转折处使用

### 2. 视线 / 方向连接（Eyeline match）

- 角色 A 看向画外左 → 切到 A 视线落点（画内左侧）
- 角色 A 抬头 → 切到 A 看到的天花板 / 招牌
- 视线方向必须连贯，不能打 180 度线

### 3. 动作 / 物体连接（Match on action）

- 钱夫人抬手 → 切到郭芙蓉抬眉（同一个"抬"的运动）
- 一只茶碗摔下 → 切到地面茶碗碎片（同一个物体）

### 非法连接（容易出戏）

- 同景别同角度连续 2 镜（看起来像跳剪故障）
- 视线方向突变 180 度
- 没有动作 / 物体衔接的硬切（除非故意制造蒙太奇冲击）

## 五、对白场景武器库（重点 / 山音融合）

**铁律**：r2v 对白场景中，**至少 30% 不能用「图1+图2 中景正反打」**。
用下面 5 种武器轮换。

### 武器 1 — 单镜跟拍（One-take walk-and-talk）

两个角色边走边说，摄影机跟拍。`r2v` 一镜到底。

prompt 模板：
```
全景跟拍, 图1 <A> 与 图2 <B> 并肩沿 <环境> 行走, 两人交替对话
"<A 台词>" "<B 台词>" "<A 台词2>", 镜头随两人缓慢横移. <mood_anchor>
```

适用：信息量大但情感平稳的对话；交代关系；走过场。

### 武器 2 — 过肩 + POV 切换（Over-the-shoulder + POV）

第一镜过肩拍 A 看 B，第二镜过肩拍 B 看 A。注意是**两个 r2v shot**，
不是一镜。

prompt 模板：
```
shot 1: 过肩中景, 视点位于 图1 <A> 右肩后方, 看向 图2 <B>,
        <B> 抬头说 "<台词>". <mood_anchor>
shot 2: 紧接前一镜头, 过肩中景反向, 视点位于 图2 <B> 左肩后方,
        看向 图1 <A>, <A> 回应 "<台词>". <mood_anchor>
```

适用：情感对峙；信息交锋；权力博弈。比正反打更"在场"。

### 武器 3 — 镜面反射（Mirror reflection）

对话双方中的一方通过镜面 / 反射面被拍到，另一方在前景实拍。

prompt 模板：
```
中景, 前景 图1 <A> 背对镜头梳头, 镜中倒映 图2 <B> 站在身后,
两人通过镜中眼神交流, <A> 说 "<台词>", <B> 沉默微微点头. <mood_anchor>
```

适用：心理隔阂；私密时刻；两人关系的隐喻。

### 武器 4 — 动作中对话（Talk-while-doing）

一方在做事（切菜 / 整理 / 走动），另一方旁观或协助，对话穿插在动作里。

prompt 模板：
```
中近景, 图1 <A> 在 <动作: 擦桌子>, 图2 <B> 倚在门边,
<A> 边擦桌子边说 "<台词>", <B> 边玩手中物件边回 "<台词>",
两人都不直视对方. <mood_anchor>
```

适用：日常感；潜台词丰富的对话；"嘴上一套心里一套"的暗流。

### 武器 5 — 旁白错位（Audio-visual offset）

一方说话，画面给另一方的反应特写（或环境空镜）。

prompt 模板：
```
大特写, 图2 <B> 的表情从平静缓慢转向震惊, 画外 图1 <A> 的台词
"<台词>" 渐入. <mood_anchor>
```

适用：信息冲击；情绪转折；台词的重量需要落在对方反应上。

### 配额表

每个有对白的 scene，按下表分配武器：

| 对话 shot 数 | 武器 1 | 武器 2 | 武器 3 | 武器 4 | 武器 5 | 正反打 |
|--------------|--------|--------|--------|--------|--------|--------|
| 2 shots | — | 1 | — | — | 1 | 0 |
| 3 shots | 1 | 1 | — | — | 1 | 0 |
| 4 shots | 1 | 1 | — | 1 | 1 | 0 |
| 5+ shots | 至少 3 种武器，正反打不超过 30% |

**自检**：把整集的 r2v dialog shot 数出来，里面用「图1+图2 中景
正反打」结构的占比 ≤ 30%。否则重新分配武器。

## 六、Double Check 三层（输出前自检）

storyboard.json 写完之后，跑一遍这三层检查：

### 第一层：导演影调一致性

- 所有 shot 的 prompt 是否都遵循 direction.json 的七项决策？
- 光线 / 色彩 / 景别比例 / 运镜偏好是否前后一致？
- 不一致的 shot 列出来，要么改 prompt 要么改 direction（不要硬撑）。

### 第二层：本段节奏匹配

- 起 / 承 / 转 / 合 每段的 shot 时长分配是否符合 dual_pacing？
- 高潮段是否有足够的"短而紧"的镜头？
- 呼吸点是否保留？

### 第三层：每场戏 ~20% 出彩设计

- 每个 scene 内统计出彩 shot 占比（非常规景别 / 运镜 / 细节）。
- 出彩点是否落在叙事重心？还是浪费在开场建场镜上？
- 出彩比 < 10% → 重新设计本场镜头（"全场都是中景正反打"问题）。
- 出彩比 10%-20% → 至少把高潮 shot 升级到出彩。
- 出彩比 ≥ 20% 且分布在叙事重心 → 合格。

## 七、常见反模式（不要做）

1. **全程中景正反打** —— 见对白武器库。
2. **每个 shot 同景别** —— 节奏单调，剪辑节奏感丧失。
3. **动作没有反应镜** —— 动作落空，观众无法消化情绪。
4. **跳级连接滥用** —— 全 → 特跳级要节制，过度使用反而失去冲击力。
5. **运镜随机** —— 全集运镜没有签名，看上去像没导演。
6. **意象不落地** —— `imagery_system.motifs` 写了但 prompt 里只字未提。
7. **空话叙事目的** —— 每个 shot 都写"展现冲突"。
